# SmartBin : A Reverse Vending Eco-Robot

## Overview
SmartBin is an AI-powered reverse vending eco-robot designed for sustainable green campuses. The system automatically detects and segregates waste materials using computer vision and embedded systems. It encourages proper waste disposal through a reward mechanism and smart monitoring features.

---

## Features
- AI-based waste detection
- Automatic waste segregation
- Reverse vending mechanism
- Reward point system
- Smart monitoring system
- Mobile app integration
- Sustainable waste management

## Working Principle
1. User inserts waste item into SmartBin.
2. Camera captures the waste image.
3. YOLOv11 model detects waste type.
4. Arduino controls motors for segregation.
5. Waste is automatically sorted.
6. Reward points are generated for the user.
7. Data is stored in Firebase for monitoring.
